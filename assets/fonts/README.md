# Fonts

Drop these TTF files here before building:

- `SpaceGrotesk-Bold.ttf`
- `SpaceGrotesk-Medium.ttf`
- `Inter-Regular.ttf`
- `Inter-SemiBold.ttf`

Sources:
- Space Grotesk: https://fonts.google.com/specimen/Space+Grotesk
- Inter: https://fonts.google.com/specimen/Inter

Referenced from `app.json` (expo-font plugin) and loaded in `src/App.tsx`.
